# 🎲 Dice Challenge Game

A clean, interactive, and responsive web-based dice game where players compete against each other to see who can roll the highest number!

## 📝 Description

The **Dice Challenge** is a lighthearted, two-player browser game built to practice core front-end development skills. Instead of forcing players to manually reload the browser, this version features an interactive button that instantly generates random numbers, updates the dice faces, and declares the winner on demand.

### Why I Built This:
* To master event listeners and DOM manipulation techniques.
* To practice writing clean, modern JavaScript logic.
* To build fluid, responsive layouts using CSS.

---

## ✨ Features

* **Interactive Roll Button:** Tap or click to roll the dice instantly without reloading the browser window.
* **Dynamic Roll Logic:** Generates true random numbers for both dice on every click.
* **Real-Time Winner Updates:** Automatically updates the header text to announce Player 1, Player 2, or a Draw.
* **Modern UI:** Clean, minimalist, and visually engaging design.
* **Fully Responsive:** Optimised for smooth gameplay on desktops, tablets, and smartphones.

---

## 🛠️ Technologies Used

* **HTML5** – Semantically structured content layout.
* **CSS3** – Custom styling, layout positioning, and typography.
* **JavaScript (ES6+)** – Logic control, event handling, and active DOM manipulation.

---

## 🕹️ How to Play

1. Open the **[Live Demo Link](https://github.io)**.
2. Click the **"Roll Dice"** (or Refresh) button at the bottom of the screen.
3. Look at the dice faces! The player with the higher number wins the round.
4. The title banner at the top will instantly display the winner!
